# 日野市 ごみ出しナビ

日野市の家庭ごみ分別・収集日を調べる単体HTMLツール。
品目名で検索すると分別区分・次回収集日・出し方の要点を表示する。

- データ出典: 東京都オープンデータ「日野市家庭ごみ分別辞典」、日野市公式サイト
- 対応スケジュール期間: 2026年5月〜12月
- 外部依存: Google Fonts のみ（それ以外は完全にオフラインで動作）

## 収集カレンダーの仕組み
`index.html`内の`SCHEDULE`は、公式カレンダーから見つけた規則性（月・火・木・金は隔週のA週/B週ローテーション、水曜は独立した4週ローテーション）をもとに`buildSchedule()`で生成している。祝日名・市内一斉清掃・年末年始休止などローテーションで導出できない情報は`HOLIDAYS` / `SPECIAL_EVENTS` / `REMARKS`に個別に記載する。

生成結果が公式データ（2026-05-04〜2026-12-28、156行）と完全一致することは以下で検証できる。

```
node scripts/verify-schedule.mjs
```

来年以降にスケジュールを延長する際は：
1. 市の公式カレンダー（新年度分）を確認し、ローテーションの位相がそのまま続いているか照合する。
2. 新しい祝日・市内一斉清掃日・年末年始休止などを`HOLIDAYS` / `SPECIAL_EVENTS` / `REMARKS`に追記する。
3. `buildSchedule()`の呼び出し範囲（終了日）を延長する。
4. 公式データから新しい`reference-schedule-*.json`を作り、`scripts/verify-schedule.mjs`で突き合わせてから反映する。
